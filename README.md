# nodebb-plugin-sso-eveonline

This [NodeBB](https://nodebb.org/) plugin allows users login/register via their [EVE Online Account](http://www.eveonline.com/) account.

## Install

    $ npm install eve-scout/nodebb-plugin-sso-eveonline
